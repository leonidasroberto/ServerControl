# ServerControl:
Gerenciador básico para instalação e ativação do xamp em servidores linux.

# Utilização:
```sh
git clone https://github.com/leonidasroberto/ServerControl;cd ServerControl;sudo chmod 755 server.sh menu.sh;./menu.sh
```
