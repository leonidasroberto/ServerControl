# ServerControl:
Gerenciador básico para instalação e ativação do xamp em servidores linux.

# Instalação:
<p>git clone https://github.com/leonidasroberto/ServerControl</p>
<p>sudo chmod 755 server</p>
<p>sudo chmod 755 server2</p>
<p>Executar: ./server</p>
